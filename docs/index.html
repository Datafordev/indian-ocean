readdirExclude(dirPath, excludes, [includeFullPath], callback)

Get a list of the files in a directory that do not have the selected extentions.

parameter type description
dirPath String

the directory to read from
excludes Array<String> or String or Array<RegExp> or RegExp

the file extention(s) to exclude or RegExp matching file name
includeFullPath [Boolean] (default false)

return the dirPath in the result
callback Function

the callback that will accept the filtered files, takes an optional error and an array of file names

Examples

io.readdirExclude('path/to/files', 'csv', function(err, files){
  console.log(files) // ['data-0.tsv', 'data-0.json']
})

io.readdirExclude('path/to/files', ['csv', 'tsv'], true, function(err, files){
  console.log(files) // ['path/to/files/data-0.json']
})

appendData(fileName, data, callback)

Append to an existing data object, creating a new file if one does not exist. For tabular formats, data must be an array of flat objects (cannot contain nested objects or arrays).

Supported formats:

  • .json Array of objects
  • .csv Comma-separated
  • .tsv Tab-separated
  • .psv Pipe-separated

Note: Does not currently support .dbf files.

parameter type description
fileName String

the name of the file
data Object

the data to write
callback Function

callback that takes an error, if any

Examples

io.writeAppendData('path/to/data.json', jsonData, function(err){
  console.log(err)
})

io.writeAppendData('path/to/data.csv', flatJsonData, function(err){
  console.log(err)
})
